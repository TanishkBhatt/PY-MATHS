------------------- SINE & COSINE WAVES -------------------

CONCEPT : PERIODIC TRIGONOMETRIC FUNCTIONS

BASE IDEA :
    From the Unit Circle (radius = 1)

DEFINITION :
    sin(x) → Vertical coordinate
    cos(x) → Horizontal coordinate

BASE VALUES :
    sin(0) = 0
    cos(0) = 1

FORM :
    y = sin(x)
    y = cos(x)

PROPERTIES :
    Period = 2π
    Range = [-1, 1]
    Amplitude = 1

RELATION :
    cos(x) = sin(x + π/2)
